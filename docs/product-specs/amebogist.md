# AmeboGist Product Specification

## Overview
News and entertainment platform for Nigerian audience.

## Features

### Core Features
1. **News Aggregation**: Latest news from various sources
2. **Categories**: Tech, Lifestyle, Politics, Entertainment
3. **User Accounts**: Save favorites, comment, share
4. **Ad Integration**: Google AdSense, direct ads

### Technical Features
1. **SEO Optimized**: Meta tags, sitemaps, structured data
2. **Performance**: Image optimization, caching
3. **Monetization**: Multiple ad slots, subscription options

## Tech Stack
- Next.js 13 (App Router)
- MongoDB for content storage
- Cloudinary for images
- Google AdSense

## Pages
1. Homepage - Latest news
2. Category pages
3. Article detail pages
4. Search results
5. User dashboard

## API Endpoints
- `/api/posts` - CRUD operations
- `/api/categories` - Get categories
- `/api/comments` - Comment management
- `/api/ads` - Ad serving

## Revenue Model
1. Ad revenue (Google AdSense)
2. Sponsored content
3. Premium subscriptions
