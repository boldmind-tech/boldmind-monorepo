# EduCenter Product Specification

## Overview
Educational platform for Nigerian students.

## Features

### Core Features
1. **JAMB Practice**: Past questions, mock exams
2. **Course Library**: Educational content
3. **Progress Tracking**: Student performance analytics
4. **Subscription**: Premium content access

### Technical Features
1. **Payment Integration**: Paystack for Nigerian payments
2. **Video Streaming**: Educational videos
3. **Quiz System**: Interactive quizzes with scoring

## Tech Stack
- Next.js 13
- Supabase Auth
- MongoDB
- Paystack API

## Pages
1. Homepage
2. Course catalog
3. JAMB practice section
4. Student dashboard
5. Payment pages

## API Endpoints
- `/api/courses` - Course management
- `/api/questions` - Question bank
- `/api/payments` - Payment processing
- `/api/progress` - Student progress

## Revenue Model
1. Course subscriptions
2. JAMB practice packages
3. School partnerships
