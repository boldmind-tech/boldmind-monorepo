# BoldMind OS Product Specification

## Overview
Personal productivity OS for individuals with ADHD and dyslexia.

## Features

### Core Brains
1. **Capture Brain**: Voice notes, quick capture
2. **Focus Brain**: Pomodoro timer, task management
3. **Connect Brain**: Knowledge graph, note linking
4. **Create Brain**: Content creation pipeline
5. **Reflect Brain**: Analytics, progress tracking

### Accessibility Features
1. **Dyslexia Mode**: OpenDyslexic font, color schemes
2. **ADHD Mode**: Distraction blocking, focus sessions
3. **Voice Interface**: Voice commands, dictation

## Tech Stack
- Next.js 13
- OpenAI Whisper API
- N8N workflows
- PostgreSQL

## Pages

- `/api/notes` - Note taking
- `/api/ai/transcribe` - Voice-to-text
